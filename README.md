# psy-client-cards

Локальное приложение для психолога для ведения карт клиентов в формате `.docx`.

Работает только локально через Streamlit на `http://127.0.0.1:8501`.

## Что реализовано (Этап 1)

- Структура проекта и автоматическое создание папок при запуске
- Модуль относительных путей через `pathlib.Path`
- Просмотр клиентских папок из `data/clients/`
- Просмотр файлов выбранного клиента
- Предупреждения о файлах, лежащих вне правильной структуры
- Подсказки по правильному размещению файлов

## Что пока не реализовано

- Обработка `.docx` (чтение, запись, шаблоны)
- LLM-интеграция (Claude, OpenAI)
- Голосовой ввод и speech-to-text
- Preview изменений и backup
- Валидация заметок и парсинг клиентов

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app.py
```

## Структура папок

```
psy-client-cards/
├─ app.py                  # Главное приложение
├─ requirements.txt
├─ .env.example
├─ prompts/                # Промпт-файлы для LLM
├─ data/
│  ├─ clients/             # Папки клиентов (anna_2332/, ...)
│  ├─ templates/           # Шаблон карты (card_template.docx)
│  ├─ inbox_notes/         # Входящие заметки
│  ├─ audio_archive/       # Аудиозаписи
│  ├─ backups/             # Бэкапы карт
│  └─ logs/                # Логи
├─ src/                    # Исходный код модулей
└─ tests/                  # Тесты
```

## Куда класть файлы

| Что | Куда |
|---|---|
| Карты клиентов | `data/clients/<client_id>/Имя 1234.docx` |
| Шаблон карты | `data/templates/card_template.docx` |
| Промпт-файлы | `prompts/` |
