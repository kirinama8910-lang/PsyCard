Заметка психолога

Центральная рабочая область для вставки заметок, выбора типа задачи и запуска пайплайна: валидация клиента → поиск/создание карты → LLM → preview → backup → сохранение. Правая панель показывает карту в структуре реального документа.

Режим: первичный приём Клиент не определён Карта не загружена

Рабочая заметка

Первая строка должна быть в формате Имя 1234. Это ключ клиента для поиска или создания карты.
В коде уже оставлены точки подключения к backend: findOrCreateClientCard, sendTextToLLM, getPreview, createBackup, saveChanges, downloadDocument, simulateAnalysis.

Операции

Эти действия оставлены для реальной интеграции с файловой логикой проекта.
Для backend здесь ожидаются безопасные операции: создание новой карты из шаблона, резервное копирование, preview diff, запись только нужных секций и safe write через временный файл.
